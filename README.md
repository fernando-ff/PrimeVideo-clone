## :checkered_flag: Starting ##

```bash
# Clone this project
$ git clone https://github.com/fernando-ff/primevideo-clone

# Access
$ cd primevideo-clone

# Install dependencies
$ npm install

# Run the project
$ npm start

# The server will initialize in the <http://localhost:3000>
```
